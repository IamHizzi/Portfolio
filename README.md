# Khizer Hayat | Portfolio

A world-class, cinematic, animated portfolio website for a Senior Full-Stack & Machine Learning Engineer.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js / React Three Fiber
- **UI Components**: Shadcn/UI patterns
- **Deployment**: Vercel-ready

## Features

- Cinematic hero section with 3D neural network animation
- Scroll-triggered animations throughout
- Parallax effects and smooth transitions
- Glassmorphism and modern design patterns
- Fully responsive (mobile-first)
- SEO optimized with OpenGraph support
- Premium dark theme

## Sections

1. **Hero** - Animated 3D background with neural network visualization
2. **About** - Story-driven introduction with achievement cards
3. **Skills** - Visual skill categories with animated progress bars
4. **Full-Stack Projects** - 5 enterprise-grade web application showcases
5. **AI/ML Projects** - 5 cutting-edge machine learning system showcases
6. **Contact** - Animated contact form with social links

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This project is configured for Vercel deployment. Simply connect your GitHub repository to Vercel for automatic deployments.
